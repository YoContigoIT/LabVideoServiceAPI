## Requisitos
NodeJS v18 (cualquier subversión)
## Descripción

API de video llamadas para video denuncia.

## Instalación

```bash
$ npm i
```

## Ejecutar la app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
